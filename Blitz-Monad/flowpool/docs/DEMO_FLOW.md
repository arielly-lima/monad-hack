# Demo Flow

1. Start on the landing page and explain the thesis: one Pix inflow becomes programmable liquidity.
2. Launch the app and connect a wallet.
3. Switch to Monad Testnet and highlight MON balance plus network badge.
4. Open Add Funds, create a Pix charge, and confirm it.
5. Show the payment timeline: generated, received, pool funded, ready for usage.
6. Move to Dashboard and highlight pool split, health, live activity, and recharge state.
7. Trigger API usage and an agent task.
8. Drop available liquidity under the threshold and show recharge armed or pending.
9. Simulate yield and show ledger credits.
10. Execute the onchain deposit action and open the explorer link.
