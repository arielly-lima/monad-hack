# Frontend

## Routes

- `/` landing page
- `/dashboard`
- `/add-funds`
- `/pool`
- `/usage`
- `/agents`
- `/ledger`

## Design system

- Dark default with deep charcoal surfaces
- Cyan and green for liquidity and success
- Amber for warnings
- Large metrics, crisp labels, and compact operational copy
- Motion used for metric reveal, feed updates, and transaction states

## Main shell

The app shell provides nav, wallet state, Monad Testnet badge, MON balance, quick actions, and a persistent activity shortcut.
